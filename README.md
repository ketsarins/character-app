# character-app

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
Note: The 'scripts' in package.json you can specify port like below comand (default: 8080).
```
--port 4000
```
### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
